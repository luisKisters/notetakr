# NoteTakr — Ideas

Running list of product ideas. Bigger explorations get their own file in
`design/ideas/` (e.g. `participants-crm.md`).

- **Global shortcut to start transcribing** — a hotkey that immediately starts
  transcription, without first opening the floating window or touching the UI.
  Complements the existing window-toggle hotkey (⌃⌥⌘N): one shortcut to show
  the note, one to just start capturing.
